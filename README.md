# Guia de Instalação do kmagnifier

Este documento explica como instalar o kmagnifier (também conhecido como KMag) no Ubuntu.
## 1. Configurar/Instalar/usar o Git [1] 

Instale o Git e configure seu nome e e-mail:

```bash
sudo apt-get install git
git config --global user.name "Seu Nome"
git config --global user.email "seu@email"
```
## 2. Instalar o kmagnifier no Ubuntu

Use o gerenciador de pacotes para instalar o programa:

```bash
sudo apt-get update
sudo apt-get install kmag
```

Após a instalação, abra o aplicativo pelo menu ou executando `kmag` no terminal.
